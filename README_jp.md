### :space_invader: &nbsp;僕について

ラサール大学オアハカ校に通っています。 \
技術やイノベーションに対して深い情熱を持っています。 \
自由な時間にパソコンやモバイル機器の保守と修理を行う技術者として働いています。 \
最先端のソリューションを適用するための新しい挑戦を常に探しています。 \
現在、ソフトウェアアーキテクチャ、開発、品質について研究しています。 \
学問的な知識と実践的な経験を融合させています。 \
メキシコに住んでおり、テクノロジーのキャリアを進めることに情熱を注いでいます。
